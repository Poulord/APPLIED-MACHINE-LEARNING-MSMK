Introducción a la Clasificación

El proceso de predecir una etiqueta de clase discreta para una instancia de datos. Es como ponerle una etiqueta a una cosa.

1. Concepto Central

La clasificación se utiliza cuando la variable de salida es categórica. El modelo aprende de datos etiquetados (datos de entrenamiento) para asignar la categoría correcta a nuevos datos.

Ejemplo Clave:

Identificar si un correo electrónico es SPAM o No SPAM.

2. Tipos: Binario vs. Multiclase

Clasificación Binaria

Predice entre solo dos posibles clases o resultados mutuamente excluyentes (0 o 1). Es la forma más sencilla de clasificación.

Ejemplo: ¿Tiene el paciente la enfermedad (Sí/No)?

3. Algoritmos Comunes: ¿Cómo se Clasifica?

La Clasificación por Proximidad

Un algoritmo simple y perezoso (lazy) que clasifica un nuevo punto basándose en la mayoría de votos de sus K vecinos más cercanos en el espacio de características.

Clave: La elección de K (un hiperparámetro) es crucial. Un K pequeño hace el modelo sensible al ruido (alta varianza), mientras que un K grande puede ignorar detalles importantes (alto sesgo).

4. El Flujo Típico de Clasificación

1

Entrenamiento

El modelo aprende de los datos históricos (con etiquetas conocidas). Ajuste de Parámetros.

2

Predicción

Se ingresan nuevos datos (sin etiqueta) para que el modelo asigne una clase.

3

Evaluación

Se mide el rendimiento del modelo con métricas (e.g., Accuracy, F1-Score, Matriz de Confusión).
