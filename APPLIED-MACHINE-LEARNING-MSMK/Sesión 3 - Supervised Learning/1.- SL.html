Aprendizaje Supervisado: Clasificación vs. Regresión

El modelo aprende de datos etiquetados para predecir salidas futuras. El tipo de salida define la tarea.

1. Clasificación

El objetivo es predecir una etiqueta categórica o discreta. El resultado siempre es una de las clases definidas en el entrenamiento.

Visualización de Clasificación Binaria

Ejemplos Comunes:

  • Detección de Spam: ¿Es Spam (Clase A) o No-Spam (Clase B)?
  • Diagnóstico Médico: ¿Riesgo Alto, Medio o Bajo? (Clasificación Multiclase)
  • Análisis de Sentimiento: ¿La reseña es Positiva, Negativa o Neutral?
  • Detección de Fraude: ¿La transacción es Fraude (Sí) o Legítima (No)?

2. Regresión

El objetivo es predecir un valor numérico o continuo. El resultado puede ser cualquier número dentro de un rango.

Visualización de Regresión Lineal

Ejemplos Comunes:

  • Predicción de Precios: Estimar el valor exacto de una vivienda ($250,000 USD).
  • Previsión Meteorológica: Predecir la temperatura exacta de mañana (28.3 °C).
  • Antigüedad del Equipo: Estimar la vida útil restante de una máquina (en meses o años).
  • Puntuación de Examen: Predecir la calificación final continua (ej. 85.5%).
