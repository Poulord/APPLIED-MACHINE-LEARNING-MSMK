FUNDAMENTOS DE MACHINE LEARNING

Top 5 Algoritmos de Aprendizaje Supervisado

Los pilares del ML: herramientas esenciales para la clasificación y la predicción.

1. Regresión Logística

Descripción: Modela la probabilidad de que una instancia pertenezca a una clase (clasificación binaria) usando la función sigmoide. El resultado se mapea entre 0 y 1.

Uso Principal: Clasificación Binaria (Ej. ¿El cliente comprará? Sí/No).

2. SVM (Máquinas de Vectores de Soporte)

Descripción: Crea un hiperplano (límite de decisión) que maximiza el margen entre las clases. Es muy efectivo en espacios de alta dimensión.

Uso Principal: Clasificación de texto, reconocimiento de patrones e imágenes.

3. Árboles de Decisión

Descripción: Utiliza una estructura jerárquica de reglas sí/no para llegar a una predicción (una clase o un valor continuo). Es fácil de entender e interpretar.

Uso Principal: Modelos que requieren alta interpretabilidad y análisis de decisiones.

4. K-NN (K-Vecinos Más Cercanos)

Descripción: Un algoritmo basado en instancias que clasifica un nuevo punto basándose en la mayoría de votos (Clasificación) o el promedio (Regresión) de sus K vecinos más cercanos.

Uso Principal: Sistemas de recomendación y problemas donde la estructura local de los datos es importante.
⚠️ ¡Advertencia! No confundir con K-Means. K-NN es un algoritmo de Aprendizaje Supervisado (usa etiquetas para clasificar), mientras que K-Means es No Supervisado (se usa para agrupar datos sin etiquetas).

5. Redes Neuronales Artificiales (ANN)

Descripción: Inspirado en el cerebro, utiliza capas de nodos interconectados (neuronas) para aprender patrones complejos y no lineales en grandes volúmenes de datos.

Uso Principal: Deep Learning (imágenes, voz, texto) y problemas con relaciones muy complejas.
Cada uno de estos métodos utiliza diferentes estrategias de optimización (como el algoritmo de Backpropagation, que se usó en su lista) para ajustar sus parámetros y minimizar el error en las predicciones.
